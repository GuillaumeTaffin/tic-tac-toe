# Tic Tac Toe game !

A Tic Tac Toe game to run in the browser, made from plain old HTML, CSS and Javascript for learning purpose ! :p

## How to play ?
Follow the steps :
- clone this repo
- open the index.html file in your favorite browser
- enjoy !
